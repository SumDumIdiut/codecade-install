# codecade-install

One-command installer for the [codecade.co.za](https://codecade.co.za) self-hosted stack:
[temutalk](https://github.com/SumDumIdiut/temutalk), [git-forge](https://github.com/SumDumIdiut/git-forge),
the `tag` game's relay/server-directory service, and a small landing portal tying them all
together behind one Cloudflare Tunnel.

## Quick start

```bash
curl -fsSL https://raw.githubusercontent.com/SumDumIdiut/codecade-install/main/install.sh -o install.sh
bash install.sh
```

On first run this clones/updates `temutalk`, `git-forge`, and `tag` as sibling directories next
to `install.sh`, generates the portal (it has no repo of its own — this script is its only
source), installs all dependencies, and drops you into an arrow-key TUI to start/stop everything
and manage the shared tunnel.

```bash
bash install.sh                # first run: clone, install, setup, then TUI
bash install.sh start all      # non-interactive start (used by the TUI itself)
bash install.sh status         # JSON status snapshot
bash install.sh bundle <dest>  # copy install.sh + all repos to <dest> (e.g. a mounted
                                # USB drive) so it can be plugged into any machine and
                                # run without needing network access to re-clone
```

Safe to run repeatedly — every step is idempotent. The TUI also does a quiet background
check for repo updates every 30 minutes and surfaces a notice (it never auto-pulls or
restarts anything without you explicitly choosing "Check for updates").

## What it manages

| Service | Path prefix | Source |
|---|---|---|
| TemuTalk | `/temutalk` | own repo, cloned |
| Forge | `/forge` | own repo, cloned |
| Tag relay | `/tag` | `tag` repo's `relay-server/`, cloned |
| Portal | `/` | generated by this script — no repo of its own |

All four run behind one Cloudflare Tunnel (reusing `temutalk`'s existing tunnel credentials)
pointed at `codecade.co.za`.
